Automatic File Uploader

Automatic file upload to hosters with API

This project lets you easily upload your files to various platforms via their API.

Name Max file size Expire in Upload
Gofile (https://gofile.io) infinite infinite
Litterbox (https://litterbox.catbox.moe) 1GB 24h
File.io (https://file.io) 2GB infinite
TmpFiles.org (https://tmpfiles.org) 100MB 1h
0x0.st (https://0x0.st/) 512MB depends on the size of your file
c-v.sh (https://c-v.sh/) 4GB depends on the size of your file
ki.tc (https://ki.tc/) 400MB depends on the size of your file
Oshi.at (https://oshi.at/) 1GB 2h
Filebin (https://filebin.net/) infinite infinite
transfer.sh (https://transfer.sh/) 10GB 14d
FroCDN (https://frocdn.com/) 15GB infinite
